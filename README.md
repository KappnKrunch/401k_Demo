# 401(k) Contribution Management App

A full-stack web application for managing 401(k) contribution settings with real-time impact visualization.

## Features

- Set contribution type (percentage or fixed amount)
- Adjust contribution rate with slider or manual input
- View Year-to-Date contributions
- See retirement impact projections
- Persistent settings storage

## Tech Stack

- **Frontend**: React 18, Axios for API calls
- **Backend**: Node.js, Express.js
- **Database**: SQLite (in-memory for simplicity)
- **Styling**: Pure CSS with modern design
